## Getting Started

1. Clone the repo
   ```sh
   git clone https://github.com/alfiyansyahhh/kasir-web-Reactjs
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run project
   ```sh 
   npm start
   ```
